# linkedin-post Skill

**Purpose:** Create REAL LinkedIn posts using browser automation.

**Usage:**
This skill automates logging into LinkedIn and creating a new text post.

**Arguments:**
*   `post_content`: The text content of the LinkedIn post.

**Environment Variables:**
*   `LINKEDIN_EMAIL`: Your LinkedIn login email.
*   `LINKEDIN_PASSWORD`: Your LinkedIn login password.

**Example Invocation:**
```bash
python scripts/post_linkedin.py "Hello from Claude Code! #AI #Automation"
```
